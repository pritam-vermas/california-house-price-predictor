## Overview

This project predicts California house prices using a Machine Learning model built with Linear Regression. The model estimates house values based on housing and location-related features such as median income, population, total rooms, total bedrooms, latitude, longitude, and ocean proximity.

The project includes data preprocessing, feature engineering, model training, evaluation, and deployment through a Gradio web application.

---

## Features

* Data Cleaning and Preprocessing
* Feature Engineering
* One-Hot Encoding
* Feature Scaling using StandardScaler
* Linear Regression Model
* Interactive Gradio Web Interface
* Public Deployment using Hugging Face Spaces
